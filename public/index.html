Workflow đề xuất

  1. Dán link hoặc nhiều link.
  2. Chọn combo video, độ dài, khổ hình và voice.
  3. Write content để viết nội dung theo đúng combo đã chọn.
  4. Tạo video để render tuần tự, tự lưu tiến trình.
  5. Xem lại video đã tạo trong Library.
App dùng Node.js thuần + JSON state. Agent có thể gọi API trực tiếp, không cần điều khiển UI.
